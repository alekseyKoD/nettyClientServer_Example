
EventLoopGroup - обрабатывает все события Netty
    (входящие/исходящие соединения, чтение/запись данных).

ServerBootstrap/Bootstrap - вспомогательные классы для настройки
    сервера и клиента соответственно.

ChannelInitializer - инициализирует конвейер обработчиков для каждого нового соединения.

StringDecoder/StringEncoder - преобразуют байты в строки и обратно.

SimpleChannelInboundHandler - базовый класс для обработки входящих сообщений.
